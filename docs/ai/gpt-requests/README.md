# GPT Requests

Use this folder to store prompts or review requests sent to GPT or other review agents.

Recommended naming:

- `YYYYMMDD-checkpoint-<letter>-request.md`
- `YYYYMMDD-pr-<number>-review-request.md`

Keep requests scoped and include:

- branch
- PR link if available
- checkpoint goal
- guardrails
- exact review questions
